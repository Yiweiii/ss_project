# Software Security Project


## Usage
*__Note:__* slices must be in *.json* format and placed in the *src/slices/* folder


To analyse all the slices:

    cd src/
    python main.py
or	
	make run
	
To analyse a specific file:
	cd src/
    python main.py -f _<filename>_
